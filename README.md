## Installation & Setup

To run the project locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/jovalmangalan/Express-portfolio.git
   ```
2. Navigate to the folder:
   ```bash
   cd Express-portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```
5. Open your browser and visit:
   ```
   http://localhost:3000
   ```
